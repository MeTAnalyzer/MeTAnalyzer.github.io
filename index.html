What is MeTAnalyzer?

MeTAnalyzer is a full stack application that analyzes realtime social media data using the IBM Watson platform and formats the results for use by the public (via the MeTAnalyzer mobile app) as well as city officials (via the MeTAnalyzer data analytics dashboard). This project was produced by Team 29 for the 2018 CUNY IBM Watson Competition, which tasked CUNY students with coming up with ideas for novel uses for the IBM Watson platform to improve the lives of New Yorkers.
